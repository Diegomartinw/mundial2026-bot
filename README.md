# 🏆 Bot Mundial 2026

Bot que envía alertas automáticas por WhatsApp sobre los partidos del Mundial FIFA 2026.

## ¿Qué hace?

- ⏰ Todos los días a las 8 AM manda el resumen de partidos del día
- 🟢 Avisa cuando arranca cada partido
- 🔴 Avisa cuando termina cada partido con el resultado final
- ⚽ Avisa cada gol en tiempo real

## Tecnologías

- Python
- API-Football (api-sports.io)
- CallMeBot (alertas por WhatsApp)
- Railway (servidor en la nube)
